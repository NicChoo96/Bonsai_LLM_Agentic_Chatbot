# Sandbox Workspace

This folder is your sandboxed workspace. Files created here are available for:

- **Prompt bootstrapping** – select files from the sidebar to include their content in your chat context
- **AI tool operations** – the AI assistant can read, write, and manage files here
- **Markdown notes** – create `.md` files to store prompts, templates, and notes

## Getting Started

1. Create new markdown files using the **New MD** button
2. Write prompt templates or reference documents
3. Select files in the sidebar to attach them to your next chat
4. Ask the AI to create, read, or modify files in this workspace
